B.	Installation (Windows)

	Python 3.10.0
	To install packages, use the command,
		py -m pip install "PackageName"
	To install specific version of the package,
		py -m pip install "PackageName==version"

C.	Installation (Unix/macOS)

	Python 3.10.0
	To install packages, use the command,
		python3 -m pip install "PackageName"
	To install specific version of the package,
		python3 -m pip install "PackageName==version"

D.	Execution

	• Install datasets from Kaggle.
		1. https://www.kaggle.com/datasets/katianakontolati/classification-of-handwritten-greek-letters
		2. https://www.kaggle.com/datasets/vrushalipatel/handwritten-greek-characters-from-gcdb

	• Update the path variable in ‘Get_Data.py’ script for both the datasets.

	• Run the Dataset_1.ipynb to fit the models on first dataset.

	• Run the Dataset_2.ipynb to fit the models on second dataset.

	• Run Exploratory_Data_Analysis.ipynb to analyse the datasets.
